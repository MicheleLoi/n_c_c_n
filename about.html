ABOUT / COLOPHON

n_c_c_n — ChaoticJazzStation — e' un set di modelli matematici che si ascoltano: sistemi dinamici e caos deterministico trasformati in suono, da suonare dal vivo come strumenti.

// pipeline

1 · Mathematica — primi prototipi: si integra il sistema (NDSolve / mappe) e si sonifica lo stato, con export WAV + MP4. Offline: si rende un file.

2 · Python — passaggio al tempo reale: fisica in NumPy, audio a bassa latenza con sounddevice, manopole live con Dear PyGui.

3 · HTML — porting nel browser con Web Audio + Canvas: niente installazione, si apre con un link. E' quello che stai usando.

build: HTML statico, zero dipendenze, zero tracking, tutto client-side.

src: canjioh/n_c_c_n  ·  audio: Web Audio API  ·  gfx: Canvas 2D
